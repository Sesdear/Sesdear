### Tools

FastAPI, mostly sync for now - async is next on the list, and I'm not going to pretend I've mastered it already. SQLAlchemy ORM for data, Postgres non-negotiable underneath it. No Alembic yet, so migrations are still a manual process. Docker keeps everything reproducible, single-stage builds for now (multi-stage is on the list, not a priority yet), Compose for local dev. GitHub Actions runs the pipeline straight through to autodeploy - no "build passes, deploy manually" half-measures. Nginx sits in front as reverse proxy, systemd runs the services underneath.
